# Kirjahahmokanta

Tietokantasovellusharjoitustyö. Tietokannassa on kirjoja ja hahmoja, jotka näissä kirjoissa esiintyvät. Sama hahmo voi olla useammassa kirjassa. Käyttäjä voi lisätä kirjoja tai hahmoja suosikeikseen.
